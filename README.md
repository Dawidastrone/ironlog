# IRONLOG — Workout Tracker with Periodization

Aplikacja do śledzenia treningów z inteligentnym silnikiem progresji opartym na 1RM.

## Funkcje
- Logowanie treningów (SET | PLAN | KG | POWT | ✓)
- 5 modeli progresji (Stały przyrost, Podwójna progresja, Periodyzacja blokowa, Autoregulacja RPE, DUP)
- Automatyczna mapa treningowa z 1RM do celu
- Korekta planu po każdym treningu
- Historia w formie tabelarycznej
- Rest timer, workout timer, volume tracker
- PWA — działa na telefonie jak natywna aplikacja

## Deploy
Hosted on Vercel. Push to `main` triggers auto-deploy.

## Tech Stack
- React 18
- Create React App
- PWA (Service Worker + Manifest)
